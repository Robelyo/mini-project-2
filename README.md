# mini-project-2
This java program models different types of goods using inheritance and can be used as a base model for customs database.
It models different types of goods such as Agricultural products, Livestock products, Resources, Manufactured goods.
It demonstrates different types of inheritance, method overriding and encapsulation.
Each item is treated as a class object that inherits shared properties from a base class.
It stores data for goods that are imported, exported, or both and automatically gives them id numbers based on what order they was stored.
Includes
goods(name, cost, import, export, ID)
food(expiry dateweight)
agricultural(crop type, origin, natural or modified)
livestock(animal type, breed, GMO status)
resources(shelf life, toxicity)
manufactured(manufacturer, model, assembly status)
idgenerator(creates unique IDs for every object automatically)
